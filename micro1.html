Introduction

The API is used to get user information in a temporary database created in form of a JSON file.p>

The API is live on https://c8m5jk.deta.dev/


About repository

This repository is created for building an API for MicroTask 1 given by HostelDevta in developement phase.I have created an express server and by using get request we are first reading the data.json file and finally in app we are calling data and exporting it as default app.


API Endpoints

Method : GET

Endpoint : '/'

This will result in "Hello"

Endpoint : "/data"

This will result in "showing the {username , email , password} from the model.json"
Data in form of JSON

The schema of the model is :

  • {
  • "artist1": {
  • "artistname" : "Deep Purple",
  • "formed" : "1968"
  • }
  • }
